# Panama Canal ArcGIS Analysis - Usage Guide

## Overview
This project contains maritime route analysis focusing on Panama Canal logistics and cost comparison.

## Project Structure
- **data/bronze**: Raw data files (CSV, XLSX)
- **data/silver**: Processed data for analysis
- **data/gold**: Final curated datasets
- **powerbi/dashboards**: Power BI dashboard files (.pbix)
- **notebooks**: Python analysis scripts
- **docs/private**: Detailed documentation and guides
- **docs/Image**: Visualization outputs and charts

## Getting Started
1. Review the documentation in `docs/private/`
2. Open Power BI dashboards from `powerbi/dashboards/`
3. Explore raw data in `data/bronze/`

## Key Files
- `powerbi/dashboards/ArcGIS.pbix`: Main Power BI dashboard
- `notebooks/panama_canal_analysis.py`: Python analysis script
- `data/bronze/Maritime_Routes_Segmented.xlsx`: Primary route data

For detailed implementation guides, see the documentation in `docs/private/`.
